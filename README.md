## Module Wise Functionality

### Customer
* Login/Registration
* View Issue
* Create/ Delete Issue
* Reopen issue

### Operator
* Login
* View Issue by id/ type
* View Customer by Firstname/Email/Id
* Create Solution
* View Solution by Id, Issue Id.

### Admin 
* Login/Register
* Create Operator
* Create Department
* View Department
* View Employee
